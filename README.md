# SWS Document Management Dashboard

A modern MERN stack document management dashboard built using:

- React + Vite
- Express.js
- MongoDB
- Node.js

## Features

- User Authentication
- JWT Authorization
- File Upload
- Dashboard UI
- Role-Based Access
- Document Management

## Tech Stack

Frontend:

- React
- Vite
- CSS

Backend:

- Node.js
- Express
- MongoDB

## Run Project

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
